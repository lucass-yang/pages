# 安装node.js

下载地址：link (opens new window) https://nodejs.org/en/download/

需要保证node.js的版本 >= 8.6

查看node.js和npm的版本号：

node -v
npm -v

2

# 配置

# 1、创建一个新的项目

mkdir vuepress
cd vuepress
1
2

# 2、使用包管理器进行初始化

npm init -y
1

# 3、安装VuePress

npm install -D vuepress
1

WARNING

不再推荐全局安装VuePress

# 4、创建基本项目结构

需要我们手动创建一下几个目录和文件

|   |-- .vuepress

|   `-- README.md
|-- package-lock.json
`-- package.json

# 5、添加启动命令

在生成的package.json文件中，添加启动命令：

"scripts: {
    "dev": "vuepress dev docs",
    "build": "vuepress build docs"
}

# 6、配置config.js

module.exports = {
	title: "Blog",
	head : [],
	themeConfig: {
		nav: [
			{text: "home", link:'/'},
			{text: "github", link: 'https://github.com'},
			{text: "blog", link: 'https://deeryang.github.io'}
		],
	sidebar: 'auto',
	}
};

# 8、在本地启动服务器

npm run dev
1

VuePress会在http://localhost:8080启动一个热重载的开发服务器

# VuePress推荐的目录结构

├── docs
